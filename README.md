# net
TCP 통신을 위한 네트워크 모듈

# 특징
- Network Interface 제공
- Boost.Asio 사용

# 빌드
- C++17 또는 최신 컴파일러 필요
- cmake 3.0 이상 버전 필요
- cmake 빌드(Visual Studio 2019 기준)
```
> cmake -G 를 사용하여 Generators 확인
> cmake ./ -G "Visual Studio 16 2019"
> 솔루션 파일(.sln) 오픈하여 빌드
```

# 요구 사항
- Nuget Package 사용하여 boost 라이브러리 설치 필요
